Đối với phần App:
Bước 1: Ta vào mục src chọn mục API vào file Config.js. Sau đó vào CMD(Terminal) gõ lệnh ipconfig để lấy được ipv4 của mạng và copy ip đó vào BASE_URL = 'http://192.168.1.27:4000/api/v1'.
Bước 2: Ta dùng lệnh npm i trong VS Code để build code.
Bước 3: Ta dùng lệnh npm start để chạy dự án. Do dự án chạy bằng Expo, bạn có thể trên máy ảo hoặc trên máy ảo android, hoặc chạy trên máy thật nhưng cần phải cài đặt phần mềm EXPO có trên CHPlay hoặc IOS.
Bước 4: Bạn vào phần mềm EXPO để quét mã QR hoặc copy mã hiển thị trong VS code sau khi start.
